# 📱 بناء وتحميل APK تلقائياً من GitHub

تم إعداد المشروع لبناء ملف APK تلقائياً عبر **GitHub Actions** عند كل تحديث.

## 🚀 الخطوات

### 1. ربط المشروع بـ GitHub
من Lovable: اضغط على **GitHub → Connect to GitHub** ثم أنشئ المستودع.

### 2. تشغيل البناء
- يعمل **تلقائياً** عند كل `push` على فرع `main`.
- أو يدوياً: اذهب إلى تبويب **Actions** في GitHub → اختر **Build Android APK** → اضغط **Run workflow**.

### 3. تحميل ملف APK
بعد انتهاء البناء (5-10 دقائق):

**الطريقة الأولى (Releases):**
- اذهب إلى صفحة **Releases** في المستودع
- ستجد آخر إصدار باسم `APK Build #N` ← حمّل `app-debug.apk`

**الطريقة الثانية (Artifacts):**
- اذهب إلى **Actions** → افتح آخر تشغيل ناجح
- في الأسفل ستجد **Artifacts** ← حمّل `app-debug-apk`

### 4. تثبيت APK على هاتفك
- انقل الملف إلى الهاتف
- فعّل **تثبيت التطبيقات من مصادر غير معروفة** في إعدادات الأمان
- افتح الملف وثبّته

## 🔧 ملاحظات
- ملف APK المُنتَج هو نسخة **Debug** (للاختبار).
- لإصدار رسمي موقّع: استخدم Android Studio (Build → Generate Signed Bundle/APK).
- أيقونة التطبيق تُولَّد تلقائياً من `resources/icon.png`.
